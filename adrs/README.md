# Architecture Decision Records

This directory contains the architecture decision records (ADRs) for `agentic-policy-engine`. Each ADR documents a significant architectural decision: the context that prompted it, the decision itself, the alternatives considered, and the consequences.

ADRs are numbered sequentially and never deleted. Superseded ADRs remain in this directory with a "Superseded by ADR-NNNN" status note.
